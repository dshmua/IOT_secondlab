# IOT_secondlab
Для запуску проекту потрібно скомпілювати обидва проекти UART_Led_GPIO_Project та UART_Project, для отримання .elf файлів.
Симуляція лаборатоного проєкту знаходиться в папці UART_Project/Proteus/Lab_2_USART.pdsprj проєкто Proteus.
Вихідний .elf файл з папки UART_Led_GPIO_Project завантажується в контроллер U2, файл з папки UART_Project в контроллер U1
